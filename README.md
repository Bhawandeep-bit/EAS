## 📌 Overview

The **Environmental Alert System** is a rule-based web application designed to monitor environmental conditions and provide real-time alerts when unsafe thresholds are exceeded.

This project focuses on improving awareness and safety by detecting abnormal environmental parameters such as temperature, humidity, and air quality.

---

## 🎯 Objective

* Build a **rule-based environmental warning system**
* Monitor environmental input data
* Trigger alerts when conditions become unsafe
* Provide a simple and user-friendly web interface

---

## ⚙️ Workflow

1. **Define Thresholds**

   * Set safe and unsafe limits for environmental parameters

2. **Monitor Input Data**

   * Collect values like temperature, humidity, and AQI

3. **Apply Rules**

   * Compare input values with predefined thresholds

4. **Trigger Alerts**

   * Generate warnings if limits are exceeded

---

## 🚀 Features

* 🌡️ Temperature monitoring
* 💧 Humidity tracking
* 🌫️ Air Quality Index (AQI) detection
* ⚠️ Real-time alert system

## 📊 Alert Rules (Example)

* Temperature > 35°C → 🔴 High Temperature Alert
* Humidity > 70% → 🟡 High Humidity Alert
* AQI > 100 → 🔴 Air Pollution Alert
* All values safe → 🟢 Environment is Safe
